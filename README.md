# Retail Sales & Customer Segmentation Analysis

## Project Overview
An end-to-end data analytics project analyzing retail sales data 
and segmenting customers using RFM Analysis and K-Means Clustering.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn)
- Power BI (Dashboard & DAX Measures)
- Machine Learning (K-Means Clustering)

## Dataset
Download from Kaggle:
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

## Customer Segments
| Segment | Customers | Avg Spend |
| Champions | 64 | $9,479 |
| Loyal Customers | 298 | $3,322 |
| At-Risk Customers | 335 | $1,669 |
| Lost Customers | 96 | $833 |

## Key Insights
- Sales grew 51% from 2014 to 2017
- Technology is highest revenue category
- 335 At-Risk customers need win-back campaign
- Furniture has lowest profit margin
